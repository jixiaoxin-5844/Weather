Weather: 一个开源的天气app
模块:

Kotlin + MVVM + Retrofit + 协程 + JetPack
用到的轮子->

    语言 ： Kotlin
    oss ： 版本更新
    和风 ： 天气展示 (使用的是API，不是AndroidSDK)  
    高德 ： 定位
    MMKV ： 缓存数据
    Retrofit ：网络请求
    城市选择 ： citypicker
    权限请求 ： permissionx
    信息收集 ： 友盟 、友盟多渠道打包
